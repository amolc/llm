## **Quick Summary**

- **Agent**: An LLM-powered system that uses tools to achieve a goal.
- **ReAct**: A foundational pattern of **Reasoning** before **Acting**.
- **Tool Calling**: Native capability of models (like Gemini) to output structured JSON for function execution.
- **Multi-Agent**: Collaborative systems (AutoGen/CrewAI) for complex problem-solving.
- **Observability**: The practice of tracing every "thought" and "action" for debugging.
